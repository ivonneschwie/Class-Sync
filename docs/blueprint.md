# **App Name**: ClassSync

## Core Features:

- Class Schedule Input: Allow students to input their class schedules, including class times, locations, and instructor names.
- Automated Conflict Detection: Automatically detect time conflicts between classes and notify the student.
- Study Group Finder: Match students with others in the same class based on their profiles and study preferences, and message each other through the app.
- AI-Powered Note Summarization: Use generative AI tool to summarize notes from lectures, creating concise study guides for each class. User provides notes, and AI summarizes them and asks clarification questions.

## Style Guidelines:

- Primary color: Soft lavender (#B5B2FF), promoting a calm and focused environment.
- Background color: Very light gray (#F0F0F5), providing a clean and distraction-free interface.
- Accent color: Pale teal (#73D5AC), for highlighting important notifications and calls to action; its refreshing brightness contrasts with the soft lavender to create visual interest.
- Body font: 'PT Sans', a humanist sans-serif offering readability and a touch of warmth.
- Headline font: 'Space Grotesk', a techy-feeling sans-serif to signal forward-thinking.
- Use clean, minimalist icons for navigation and representing different subjects.
- Employ a card-based layout for classes and schedules, making the information digestible and easy to navigate.
- Use subtle animations to provide feedback on user actions, like adding a class or confirming a study group match.
